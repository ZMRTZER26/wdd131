Overview

Purpose

Streamers often play a variety of image based games from guess who to tierlists. This site will create one such game. A game of Guess Who with VTubers as the theme. An easy to use, easy to play game that comes with the ability to add new images into the game.

Audience

The target audience of this site will be live streamers, specifically VTubers, and their communities.

Dynamic elements

1. A function that spins through a list of images before stopping on one of the images.

2. It will start with a basic start button that will activate the first function.

3. After the first spin, the image that is stopped on will act as the button.

4. An object that will hold all of the images.

5. A JS file that will call the object and run the aformentioned functions.

6. Functions that will take images added via form, add them to the object and random selection functions.

Content

Home page

On click, will spin through a selection of images, stopping on a random image. If user is unable to guess the name, the hint button can be pressed to display the name. Clicking the image will start the process over again.
